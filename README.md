This repository contains a collection of interactive mini-projects built using HTML, CSS, and Vanilla JavaScript. Each project is designed to practice and demonstrate fundamental front-end development skills such as DOM manipulation, event handling, conditional logic, and API usage. The projects include a Calculator for performing basic arithmetic operations, a Currency Converter that fetches live exchange rates via API, a Rock Paper Scissors game with random logic and visual feedback, and a Tic Tac Toe game with complete win/draw detection. These projects serve as practical examples of how core JavaScript concepts can be applied to build real-world applications. Each mini app focuses on simplicity, responsiveness, and user experience, making this repository ideal for beginners and those looking to strengthen their frontend development foundation.
